# Slide Right Modal

## What does this do?

Creates a pure CSS modal that slides in smoothly from the right with a minimalist tech-inspired design.

## How is it used?

Open `demo.html` in a browser. Click **Open Modal** to display the modal. The interaction is implemented using a hidden checkbox and CSS only.

## Why is it useful?

It provides a lightweight, responsive, keyboard-friendly modal with customizable CSS variables and no JavaScript dependency.
