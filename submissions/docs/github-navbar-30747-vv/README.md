# GitHub Repository Navigation Demo

## What does this do?

This submission demonstrates a responsive navigation bar with a GitHub button that links directly to the official EaseMotion CSS GitHub repository.

---

## How is it used?

```html
<a
    class="github-btn"
    href="https://github.com/SAPTARSHI-coder/EaseMotion-css"
    target="_blank"
    rel="noopener noreferrer">
    GitHub
</a>
```

---

## Why is it useful?

- Improves repository visibility.
- Makes it easier for contributors to access the source code.
- Follows common open-source website navigation practices.
- Enhances user experience by providing quick access to GitHub.
