# CSS Confetti / Celebration — Issue #30443

Pure CSS confetti effect triggered by toggling `.ease-confetti` class on the container.

## Classes

| Class | Purpose |
|-------|---------|
| `.ease-confetti` | Container (overflow hidden) |
| `.ease-confetti-piece` | Individual falling piece |
| `.ease-confetti-square` | Square shape |
| `.ease-confetti-circle` | Circular shape |
| `.ease-confetti-star` | Star shape (clip-path) |

## CSS Variables

- `--ease-confetti-duration` (default 3s)
- `--ease-confetti-colors` (comma-separated)
