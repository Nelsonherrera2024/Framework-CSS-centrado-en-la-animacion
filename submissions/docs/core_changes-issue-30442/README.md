# Animated Gradient Background — Issue #30442

Smoothly animated gradient backgrounds.

## Classes

| Class | Effect |
|-------|--------|
| `.ease-gradient-animate` | Animated gradient (default 6s) |
| `.ease-gradient-animate-slow` | 12s cycle |
| `.ease-gradient-animate-fast` | 3s cycle |
| `.ease-gradient-animate-angle` | Rotating angle variant |
| `.ease-gradient-animate-sunset` | Orange/red/pink |
| `.ease-gradient-animate-ocean` | Cyan/blue/indigo |
| `.ease-gradient-animate-aurora` | Green/teal/cyan |
| `.ease-gradient-animate-neon` | Pink/purple/blue |

## CSS Variables

- `--ease-ga-speed`, `--ease-ga-angle`, `--ease-ga-bg-1` through `-4`
