Gradient Text Flow

Animated gradient text via SCSS mixins

Static Gradient

Applied via ease-gradient-text-base — a fixed gradient fill with background-clip: text.

Animated Flow

Applied via ease-gradient-text-flow — the gradient slides horizontally for a shimmer effect.

SCSS usage:

@use 'mixins' as gradient;
@include gradient.ease-gradient-text-keyframes;
@include gradient.ease-gradient-text-flow;
