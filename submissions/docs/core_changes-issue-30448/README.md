# CSS Loader / Spinner Gallery — Issue #30448

Comprehensive collection of CSS-only loaders and spinners.

## Classes

| Class | Type |
|-------|------|
| `.ease-loader-spin` | Rotating circle |
| `.ease-loader-pulse` | Pulsing dot |
| `.ease-loader-dots` | Three bouncing dots |
| `.ease-loader-ripple` | Expanding rings |
| `.ease-loader-hourglass` | Rotating hourglass |
| `.ease-loader-ellipsis` | Animated ellipsis |
| `.ease-loader-sm` | Small size |
| `.ease-loader-lg` | Large size |

## CSS Variables

- `--ease-loader-size`, `--ease-loader-color`, `--ease-loader-track`, `--ease-loader-speed`
