This demo showcases the fix for two light mode visibility issues in the documentation pages:

  • The active "Buttons" tab in the navigation bar now updates its background and text colors dynamically using theme variables, ensuring high contrast in both dark and light modes.
  • The "docs-h3" heading class color now uses var(--heading-color) instead of being hardcoded to white, remaining perfectly readable when theme toggled.
