# Circular Progress Ring — Issue #30445

SVG-based circular progress ring with stroke-dasharray animation.

## Classes

| Class | Purpose |
|-------|---------|
| `.ease-progress-ring` | SVG container |
| `.ease-progress-ring-track` | Background circle |
| `.ease-progress-ring-fill` | Progress arc |
| `.ease-progress-ring-label` | Center percentage |
| `.ease-progress-ring-sm` / `-lg` | Size variants |
| `.ease-progress-ring-success` / `-warning` / `-danger` | Color variants |

## CSS Variables

- `--ease-ring-size` (default 6rem)
- `--ease-ring-stroke` (stroke width)
- `--ease-ring-color`, `--ease-ring-track`
- `--ease-ring-value` (0–100)
- `--ease-ring-duration`
