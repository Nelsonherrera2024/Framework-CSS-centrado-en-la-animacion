# Shape Morphing — Issue #30444

CSS clip-path shape morphing utility with smooth transitions.

## Classes

| Class | Shape |
|-------|-------|
| `.ease-shape-morph` | Base (circle by default) |
| `.ease-shape-circle` | Circle (50%) |
| `.ease-shape-triangle` | Triangle |
| `.ease-shape-diamond` | Diamond |
| `.ease-shape-hexagon` | Hexagon |
| `.ease-shape-star` | 5-point star |
| `.ease-shape-pentagon` | Pentagon |
| `.ease-shape-chevron` | Chevron |
| `.ease-shape-cross` | Cross |
| `.ease-shape-polygon` | Custom via --ease-shape-clip |

## CSS Variables

- `--ease-morph-duration` (default 0.4s)
- `--ease-morph-easing`
- `--ease-shape-clip` (for custom polygon)
