Scroll to top button

FAB appears after 300px scroll · fade-in + slide-up · smooth scroll to top

Demo

Scroll down — the teal button fades in at the bottom-right corner. Click it to return to top.

Behavior

  • hidden by default
  • appears at 300px scroll with fade + slide-up
  • fixed position, bottom-right
  • smooth scroll to top on click
  • hover glow effect
  • respects prefers-reduced-motion

Usage

<button id="top-btn" onclick="scrollToTop()">&#8593;</button>

<script>
window.addEventListener('scroll', function() {
  var b = document.getElementById('top-btn');
  if (window.scrollY > 300) b.classList.add('on');
  else b.classList.remove('on');
});
function scrollToTop() {
  window.scrollTo({ top: 0, behavior: 'smooth' });
}
</script>
