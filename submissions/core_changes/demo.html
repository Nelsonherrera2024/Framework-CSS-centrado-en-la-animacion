Search expand

Input expands from 40px icon to full bar on focus · CSS-only · :focus-within

🔍

How it works

The wrapper uses :focus-within to transition width from 40px to 260px. The input fades in as the wrapper expands. The icon stays fixed on the left. No JavaScript needed.

Behavior

  • collapsed: 40px circle, icon only
  • focus: expands to 260px, input fades in
  • blur: collapses back to icon
  • keyboard accessible via Tab
  • respects prefers-reduced-motion

Usage

<div class="srch">
  <span class="srch-ico">&#128269;</span>
  <input type="text" class="srch-inp" placeholder="Search..." />
</div>
