# Shimmer skeleton animation

CSS-only shimmer loading skeleton. Adds `@keyframes easeShimmer` and `.ease-shimmer` class with customizable `--shimmer-clr` variable.

- teal/cyan palette on demo page
- all demo styles in `<style>` tag
- `style.css` only contains shimmer animation
- reduced-motion support

## Files

- `demo.html` — card + text skeleton demos, usage code
- `style.css` — keyframes and class definition
- `README.md` — this file
