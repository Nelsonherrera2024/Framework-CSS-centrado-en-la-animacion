# Scroll to top button

Floating action button with fade-in + slide-up entrance after 300px scroll. Smooth-scrolls to top on click.

- fixed bottom-right position
- hidden by default, appears with `.on` class
- teal/cyan color scheme
- hover glow effect
- `prefers-reduced-motion` support
- responsive sizing

## Files

- `demo.html` — page with lorem ipsum filler, scroll down to see the button
- `style.css` — button position, visibility toggle, transitions, hover, reduced-motion
- `README.md` — this file
