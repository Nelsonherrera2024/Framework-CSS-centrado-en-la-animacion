# Expandable search input

Search input that expands from 40px icon to 260px bar on focus.

- `:focus-within` on wrapper triggers width transition
- input fades in as wrapper expands
- rose/crimson accent on focus
- reduced-motion support
- no JavaScript required

## Files

- `demo.html` — live demo, behavior docs, usage code
- `style.css` — wrapper, icon, input styles, transitions
- `README.md` — this file
