Sidebar Navigation Component

A vertical sidebar with slide-in animation for dashboards and admin panels.

Features

  • Fixed or relative vertical navigation with background
  • Individual nav links with icon and label support
  • Active state with primary color and left border accent
  • Hover background highlight
  • Collapsed mode (icons only, labels hidden)
  • Mobile slide-in animation via transform: translateX
  • Configurable width via --ease-sidebar-width custom property
  • Optional JavaScript for toggle, collapse, and mobile handling

Usage

Apply these classes to build a sidebar:

  • .ease-sidebar — The sidebar container
  • .ease-sidebar-header — Header with branding
  • .ease-sidebar-nav — Navigation items wrapper
  • .ease-sidebar-item — Individual nav link
  • .ease-sidebar-item.active — Active/highlighted item
  • .ease-sidebar-collapsed — Collapsed state
  • .ease-sidebar-overlay — Mobile overlay
  • .ease-sidebar.open — Open mobile sidebar
