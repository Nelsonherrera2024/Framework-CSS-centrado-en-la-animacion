# Glassmorphism Profile Card Hover Effect

1. What does this do? A responsive profile card with glassmorphism backdrop blur, smooth hover lift, gradient border glow on hover, and animated social media icons — all in pure HTML and CSS with zero JavaScript.

2. How is it used? Add the HTML markup and link `style.css`. Classes like `.profile-card`, `.avatar`, `.social-icon`, `.glow-border` handle the glass effect, layout, and hover animations.

3. Why is it useful? Glassmorphism is a modern UI trend used in portfolios, landing pages, and dashboards. This component demonstrates backdrop-filter blur, gradient borders, hover transitions, and responsive layout — aligning with EaseMotion CSS's animation-first philosophy.
