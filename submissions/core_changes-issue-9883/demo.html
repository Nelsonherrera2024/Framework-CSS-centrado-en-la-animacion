Issue #9883: CSS Containment Utilities

Optimize rendering performance with content-visibility and contain utility classes

1. What is CSS Containment?

CSS containment tells the browser that certain subtrees are independent from the rest of the page. The browser can then skip layout, paint, and style calculations for off-screen or isolated elements — dramatically improving initial load and scroll performance.

content-visibility: auto
Defers rendering of off-screen elements
contain: strict
Full isolation (layout + paint + size + style)
contain: content
Safe default (layout + paint + style)

Performance Tip: Use ease-content-visibility-auto on long scrollable sections (articles, feeds, lists). Use ease-contain-strict on reusable widgets and cards to prevent layout side effects. For animation-heavy components, use ease-contain-layout-paint to ensure transforms don't leak out of the element.
