# Animated Pricing Table

A CSS-only pricing comparison table with 3 plans side by side. Includes a featured/highlighted "Popular" plan with badge, checkmarks/crosses for features, and animated CTA buttons. Hover effects lift cards and highlight feature rows.

**Created by @saurabhhhcodes**

## Features

- Pure CSS — no JavaScript required
- 3 pricing plans: Starter, Professional (featured), Enterprise
- Featured plan badge with gradient styling
- Checkmark (✓) and cross (✕) feature indicators
- Card lift on hover with smooth cubic-bezier transition
- Row hover highlight on features
- Gradient CTA buttons with hover lift
- Glassmorphism design with backdrop blur
- Fully responsive grid layout
- Scale emphasis on the featured plan
