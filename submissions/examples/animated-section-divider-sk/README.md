# Animated Section Divider

CSS-only section dividers with decorative gradient lines and wave patterns.

## Features

- Six divider styles: gradient line, wave (SVG), angled clip, dotted pattern, arrow with bounce, geometric conic pattern
- Animated gradient shift on the gradient line divider
- Hover interaction on the angled clip divider
- Continuous scroll animation on the dotted pattern
- Gentle bounce animation on the arrow icon
- Wave SVG drift animation
- Dark and light section variants
- Custom properties for easy theming

## Usage

Include the `style.css` and use the BEM class structure. Each divider uses `divider--{style}` modifier. Place between content sections. The wave and arrow dividers can be placed inside dark or light backgrounds.

## Browser Support

All modern browsers (Chrome, Firefox, Safari, Edge). Uses CSS gradients, clip-path, animations, and SVG.
