Section Dividers

Six decorative divider styles to separate your content sections

Gradient Line

Minimal & Clean

A simple gradient line that stretches across the full width. Subtle and elegant.

The gradient transitions smoothly from purple to blue to pink, creating a gentle visual separation.

Wave Pattern

Organic Flow

A fluid wave shape that adds movement and personality between sections.

The wave divider uses an SVG path with a smooth bezier curve. The color inherits from the parent section.

Angled Clip

Sharp Transition

A diagonal clip-path creates a sharp, dynamic angle between sections.

Pure CSS clip-path with no images or SVGs. The angle direction can be flipped with a modifier class.

Dotted Pattern

Playful Rhythm

A repeating dotted line that creates a rhythmic, playful separation.

Uses CSS repeating gradients to create the dotted effect. The dot size and spacing are customizable via custom properties.

Arrow Divider

Directional Cue

A centered arrow icon with lines on both sides, suggesting forward motion.

↓

The arrow bounces gently to draw the eye downward, encouraging the user to continue scrolling.

Geometric Pattern

Modern Structure

A repeating geometric diamond pattern for a modern, tech-forward look.

Built with CSS repeating conic gradients. The pattern scale and colors are adjustable via custom properties.
