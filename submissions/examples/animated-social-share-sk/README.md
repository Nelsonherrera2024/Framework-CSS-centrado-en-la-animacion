# Animated Social Share

A CSS-only floating social share button bar with brand-colored platform icons (Facebook, Twitter/X, LinkedIn, Pinterest, Reddit, WhatsApp). Features a sticky/fixed sidebar positioning, hover lift effect with brand colors, and tooltip labels on hover.

**Created by @saurabhhhcodes**

## Features

- Pure CSS — no JavaScript required
- Fixed floating sidebar (stacks horizontally on mobile)
- 6 social platforms with distinct brand colors
- SVG icons for each platform
- Hover lift effect with cubic-bezier spring transition
- Tooltip labels that slide in on hover
- Active press state with scale-down
- Glassmorphism content area
- Responsive layout with horizontal stacking on mobile
