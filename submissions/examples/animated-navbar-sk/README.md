# animated-navbar-sk

Responsive navigation bar with logo, nav links, and hamburger menu using the checkbox hack for mobile. Active link highlight is applied via the `.active` class. On desktop the links are laid out horizontally; on mobile they stack vertically with a smooth slide-down animation when the hamburger is toggled. The hamburger animates into a close icon on toggle.

**Features:**
- Sticky navbar with glass-morphism background
- Checkbox-hack toggle (no JavaScript)
- Animated hamburger to close icon
- Active link styling
- Smooth mobile menu open/close
