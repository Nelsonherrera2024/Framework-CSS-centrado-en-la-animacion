New Component

ease-image-zoom-hover

A premium, highly-customizable pure CSS image hover zoom component featuring a smooth glassmorphic overlay and an animated zoom-in hint icon.

Default

Classic Zoom

Smooth 1.15x scaling transition with a glassmorphic dark overlay and zoom-in indicator icon.

--subtle

Gentle Scale

Subtle 1.07x zoom. Perfect for cleaner, more modern interfaces where minimalist transitions are desired.

--intense

Dramatic Focus

Aggressive 1.25x scaling for rich imagery, product zooms, or galleries highlighting finer details.

--slow

Slow Motion

Elongated 0.75s transition that emphasizes a luxurious, premium feel for editorial layouts.

--no-overlay

Pure Image Zoom

No dark overlay mask or search icon. Only the image zooms smoothly inside its card frame.

CSS Custom Vars

Purple 1.35x Override

Demonstrates customization via CSS variables, using a purple backdrop, custom duration, and larger scale.

Component Documentation

Integrating the component is simple. Wrap your image and overlay HTML markup with the container class .ease-image-zoom-hover. Customize behavior by appending modifiers or overriding design variables.

<div class="ease-image-zoom-hover">
  <img src="image.jpg" alt="Description" class="ease-image-zoom-img">
  <div class="ease-image-zoom-overlay">
    <svg class="ease-image-zoom-icon" aria-hidden="true">...</svg>
  </div>
</div>

Modifier Classes

Class Name Type Description
.ease-image-zoom-hover Base The core container wrapper. Enables hover-to-zoom context.
.ease-image-zoom-hover--subtle Intensity Mod Decreases the zoom scale to 1.07x.
.ease-image-zoom-hover--intense Intensity Mod Increases the zoom scale to 1.25x.
.ease-image-zoom-hover--slow Speed Mod Slows transition speed down to 0.75s.
.ease-image-zoom-hover--fast Speed Mod Speeds up transition to 0.25s.
.ease-image-zoom-hover--no-overlay Display Mod Hides the dark overlay and icon, zooming only the image.

CSS Custom Customization Variables

Custom Variable Default Value Purpose
--ease-zoom-scale 1.15 Scale factor applied to the image on hover (e.g. 1.25).
--ease-zoom-duration 0.45s Transition timing length for the scale, fade-in, and filter shifts.
--ease-zoom-ease cubic-bezier(0.25, 1, 0.5, 1) Transition easing curve for zoom and overlay fading.
--ease-zoom-overlay-bg rgba(15, 23, 42, 0.4) Overlay background overlay color and transparency.
--ease-zoom-icon-color #ffffff Fill/stroke color of the hint SVG icon.
--ease-zoom-icon-scale-init 0.7 Starting scale for the hint icon prior to hover animation.
--ease-zoom-icon-scale-hover 1 Final scale for the hint icon on hover.
