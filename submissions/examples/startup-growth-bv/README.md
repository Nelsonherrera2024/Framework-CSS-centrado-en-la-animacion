# Startup Growth Platform

An ultra-premium startup operating system and growth dashboard showing recurring revenue metrics, Cap table breakdowns, acquisition funnels, NPS trackers, OKR progress bars, and team velocity gauges. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### 16 UI Sections

| #   | Section                      | Description                                                                     |
| --- | ---------------------------- | ------------------------------------------------------------------------------- |
| 1   | **Header**                   | Startup branding with Series A funding round badge and navigation buttons       |
| 2   | **Growth Metrics Row**       | Real-time display for MRR, ARR, Monthly Burn Rate, and Runway Remaining         |
| 3   | **Acquisition Funnel**       | Interactive conversion funnel tracking traffic down to paid conversions         |
| 4   | **Investor Dashboard**       | Equity Cap Table listing shareholder distribution percentages and valuations    |
| 5   | **KPI Monitoring Grid**      | Cards showing LTV:CAC, ARPU, and CAC metrics with trend sparklines              |
| 6   | **Revenue Tracking**         | Growth forecast ledger listing target achievements and expansions               |
| 7   | **Product Metrics Panel**    | DAU, MAU, retention rates, and revenue churn trackers                           |
| 8   | **Executive BI Summary**     | Telemetry logs showing Net Revenue Retention (NRR) and gross margins            |
| 9   | **Cohort Analysis Table**    | Customer cohort retention percentages over a 6-month timeline                   |
| 10  | **User Growth Chart Area**   | Bar chart visualization indicating user growth trends across quarters           |
| 11  | **Feature Adoption Heatmap** | Color-coded grid detailing feature usage percentages                            |
| 12  | **NPS Score Gauge**          | Customer satisfaction score dial with dynamic radial filling                    |
| 13  | **Team Velocity Tracker**    | Horizontal point progress indicator for agile sprints                           |
| 14  | **Competitive Landscape**    | Side-by-side product feature comparison cards against competitor legacy systems |
| 15  | **Fundraising Timeline**     | Interactive horizontal roadmap stepper showing funding milestones               |
| 16  | **OKR Progress Tracker**     | Objective progress grids showing completion ratios for Key Results              |

## 🎨 Theme & Typography

- **Colors**: Deep Navy (`#0d1117`), Dark Slate (`#161b22`), Violet Purple (`#7c3aed`), Emerald Green (`#10b981`), Amber Orange (`#f59e0b`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft fade-in-up entrance transitions, pulse-glow borders, and hover lift effects.
