`word-break-break-word-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/word-break-break-word-tm/style.css">
<div class="word-break-break-word-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .word-break-break-word: breaks at soft wrap points

## Impact

Prevents horizontal overflow on long URLs
