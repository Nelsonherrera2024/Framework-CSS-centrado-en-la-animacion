# Floating FAB Animation

A reusable floating action button animation example for EaseMotion CSS. It demonstrates a smooth floating motion, gentle pulse glow, and a hover scale effect that feels polished and friendly.

## What it does

This utility creates a floating button that:

- slowly moves up and down with smooth easing
- shows a subtle pulse glow behind the button
- enlarges slightly on hover
- works well on mobile and desktop

## How to use

Import `style.css` and apply the utility classes to a button or link.

### HTML example

```html
<a href="#" class="fab-float">
  <span class="fab-float__inner">+</span>
</a>
```

### Notes

- The `.fab-float` class handles the button styling and hover interaction.
- The `.fab-float__inner` class contains the floating motion animation.

## Why it fits EaseMotion CSS

This animation is lightweight and reusable, using clean CSS keyframes and transforms. It aligns with EaseMotion CSS by providing a subtle, motion-driven enhancement that improves UI feedback without adding complexity.
