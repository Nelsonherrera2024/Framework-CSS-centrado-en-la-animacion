# Global Theme Marketplace Demo

### What does this do?
A simple UI demo showing theme cards for a global theme marketplace system.

### How is it used?
Use the `.card`, `.dark-theme`, and `.minimal-theme` classes in HTML to render theme previews.

### Why is it useful?
It demonstrates reusable theme-based UI patterns that align with EaseMotion CSS philosophy of modular, accessible, and animation-safe design.
