# Invert Filter Utility Showcase

Demonstrates the implementation of the CSS invert filter utilities under issue #15202.

## Invert Utilities Included

- `.ease-filter-invert`: Applies 100% invert filter to the target element.
- `.ease-filter-invert-0`: Removes any active invert filter (setting it to 0).

## Showcase Layout

Open `demo.html` in a web browser. Hovering over the card dynamically inverts the graphic/icon representation using standard transitions and `.ease-filter-invert-0`.
