ease-page-loader-progress

A lightweight CSS-only page loader displayed at the top of the page.

Default example

Scroll this demo to see the fixed loader animate across the top of the page.

Customization Example

Set CSS variables to change color, height and speed.

/* Customization example */
.custom-loader{
	--ease-loader-color:#06b6d4;
	--ease-loader-height:6px;
	--ease-loader-duration:1s;
	--ease-loader-bg:rgba(6,182,212,0.06);
}


<div class="ease-page-loader-progress demo-inline custom-loader"></div>

This demo is responsive — the loader spans the full width on narrow screens.
