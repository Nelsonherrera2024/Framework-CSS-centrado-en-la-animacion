# Notification Stack Toast

A stacked toast notification system with cascade slide-in animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--toast-bg` | `#2d2d44` | Toast background |
| `--toast-border` | `#4a4a6a` | Toast border color |
| `--stack-gap` | `8px` | Gap between toasts |
| `--slide-duration` | `0.35s` | Slide animation speed |
