# Ease Spin

A continuous 360° rotation utility for EaseMotion CSS.

## Features

✅ Infinite rotation  
✅ Smooth linear timing  
✅ Customizable speed using CSS variable  
✅ Reverse spin variant  
✅ Production-ready utility classes

---

## Usage

```html
<div class="ease-spin"></div>

<div class="ease-spin-reverse"></div>
```

## Speed Customization

```css
:root{
  --ease-spin-speed:1s;
}

.fast{
  --ease-spin-speed:.5s;
}

.slow{
  --ease-spin-speed:3s;
}
```
