# Scrollbar Color Utilities

CSS utility classes for the `scrollbar-color` property.

## Usage

```html
<div class="scrollbar-thumb-primary">...</div>
```

## Classes

- `.scrollbar-thumb-primary` — scrollbar-color: var(--ease-primary) var(--ease-bg);
- `.scrollbar-thumb-secondary` — scrollbar-color: var(--ease-secondary) var(--ease-bg);
- `.scrollbar-thumb-accent` — scrollbar-color: var(--ease-accent) var(--ease-bg);
- `.scrollbar-thumb-gray` — scrollbar-color: #64748b #f1f5f9;
- `.scrollbar-thumb-dark` — scrollbar-color: #1e293b #0f172a;
- `.scrollbar-thumb-light` — scrollbar-color: #e2e8f0 #f8fafc;
- `.scrollbar-track-primary` — scrollbar-color: var(--ease-bg) var(--ease-primary);
- `.scrollbar-track-secondary` — scrollbar-color: var(--ease-bg) var(--ease-secondary);
- `.scrollbar-track-gray` — scrollbar-color: #f1f5f9 #64748b;
- `.scrollbar-color-thumb-auto` — scrollbar-color: auto;
- `.scrollbar-color-inherit` — scrollbar-color: inherit;
- `.scrollbar-color-initial` — scrollbar-color: initial;
- `.scrollbar-color-unset` — scrollbar-color: unset;
- `.scrollbar-thumb-red` — scrollbar-color: #ef4444 transparent;
- `.scrollbar-thumb-blue` — scrollbar-color: #3b82f6 transparent;
- `.scrollbar-thumb-green` — scrollbar-color: #22c55e transparent;
- `.scrollbar-thumb-yellow` — scrollbar-color: #eab308 transparent;
- `.scrollbar-thumb-purple` — scrollbar-color: #a855f7 transparent;
- `.scrollbar-thumb-pink` — scrollbar-color: #ec4899 transparent;
- `.scrollbar-thumb-orange` — scrollbar-color: #f97316 transparent;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
