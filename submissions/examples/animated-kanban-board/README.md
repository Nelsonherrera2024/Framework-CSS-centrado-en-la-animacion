# Animated Kanban Board Component

A modern responsive Kanban board component with smooth animations, priority badges, animated card entrance effects, hover interactions, and responsive columns.

## Features

-  Responsive Kanban board layout
-  Animated card entrance
-  Priority badges (High, Medium, Low)
-  Animated task count badges
-  Hover lift animation
-  Mobile-friendly design
-  Pure HTML & CSS
-  No external libraries

## Usage

Include the stylesheet and use the provided HTML structure.

```html
<link rel="stylesheet" href="style.css">

<div class="kanban-board">
</div>
