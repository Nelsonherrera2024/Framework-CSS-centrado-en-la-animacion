# Ease Log Viewer

## What does this do?

A terminal-style log viewer that displays scrollable logs with animated entries, color-coded log levels, auto-scroll support, and a clear button.

---

## How is it used?

Open `demo.html` in any modern browser.

Example:

```html
<div class="log info ease-fade-in-up">
    <span class="level">INFO</span>
    <span>Server started successfully.</span>
</div>
```

Use the controls to:

- Add sample log entries
- Toggle auto-scroll
- Clear all logs

---

## Why is it useful?

This component is useful for dashboards, admin panels, monitoring tools, debugging interfaces, and developer consoles. It provides a clean terminal-inspired experience with subtle animations that align with the animation-first philosophy of EaseMotion CSS.
