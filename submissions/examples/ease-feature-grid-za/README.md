# CSS Feature Grid

## What does this do?
Feature card grid layouts with responsive columns, icon styles, border cards, and highlight tags.

## How is it used?
```html
<div class="fg-grid fg-col3"><div class="fg-card"><div class="fg-ico">&#9889;</div><h3>Title</h3><p>Description</p></div></div>
```

## Why is it useful?
Responsive feature grids. CSS Grid auto-fit columns, icon display, hover lift effects, highlighted card with tag.
