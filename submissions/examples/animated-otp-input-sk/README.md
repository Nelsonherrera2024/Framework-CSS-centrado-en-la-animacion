# Animated OTP Input

CSS-only OTP input with 6 digit boxes, filled states, error shake animation, and focus effects.

## Features
- 6 digit input boxes with monospace font
- Focus state with lift and glow effect
- Filled state with accent color border
- Error shake animation
- Error message with fade-in
- Responsive sizing

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`.
