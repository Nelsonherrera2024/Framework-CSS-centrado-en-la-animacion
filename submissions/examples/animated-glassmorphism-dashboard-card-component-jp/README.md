# Animated Glassmorphism Dashboard Card Component

## Overview

A premium CSS-only glassmorphism dashboard card featuring blurred backgrounds, glowing borders, smooth hover animations, and modern UI styling.

## Features

- Glassmorphism blur effect
- Smooth hover lift animation
- Glowing shadow transitions
- Responsive dashboard layout
- Modern statistics card design
- Lightweight CSS-only implementation

## Files

- demo.html – Displays the glassmorphism dashboard cards
- style.css – Contains card styles and hover animations
- README.md – Provides documentation and usage instructions

## Usage

Open demo.html in any modern browser to view the animated glassmorphism dashboard cards.

Hover over cards to experience the floating and glowing effects.
