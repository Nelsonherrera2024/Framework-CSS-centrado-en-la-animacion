# Animated FAQ Accordion

A CSS-only FAQ accordion using the checkbox hack, featuring smooth height expand/collapse via `grid-template-rows` and a rotating +/- icon. Created by @saurabhhhcodes.

## Features

- Pure CSS (no JavaScript)
- Checkbox hack for toggle behavior
- Smooth `grid-template-rows` animation for height
- Animated rotating plus/minus icon
- BEM naming conventions
- 5 FAQ items with clean, accessible design

## Usage

Include `style.css` in your project. The accordion is structured with hidden checkboxes (`.faq__toggle`), labels as questions (`.faq__question`), and the answer content (`.faq__answer`). Checking a checkbox triggers the expand animation.
