`backdrop-brightness-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/backdrop-brightness-tm/style.css">
<div class="backdrop-brightness-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .backdrop-brightness-[50|100|150|200]: brightness scale

## Impact

Enables dimming of backgrounds for focus overlays
