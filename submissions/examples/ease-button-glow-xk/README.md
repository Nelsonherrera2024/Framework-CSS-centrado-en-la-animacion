# Button Glow

1. What does this do? A neon-glow button that intensifies its border glow on hover with a subtle lift and background blend.
2. How is it used? Add class .bg-btn; set --btn-glow-color to any color. Use modifier classes like bg-btn--blue for preset colors.
3. Why is it useful? It fits EaseMotion CSS's human-readable, animation-first philosophy by making call-to-action buttons visually engaging with minimal markup.
