# Badge New Glow Animation

## What does this do?

Adds a continuous, subtle neon glow pulsing animation to badges (like 'NEW', 'SALE', or 'HOT'), drawing immediate attention to promotional or recently added elements without being distracting.

## How is it used?

Add the badge classes to your HTML element:

```html
<span class="badge badge-glow-coral">NEW</span>
<span class="badge badge-glow-purple">SALE</span>
```

## Why is it useful?

In promotional, e-commerce, or SaaS dashboards, it is critical to guide the user's attention to new features, sales, or updates. This animation builds on existing badge patterns by adding a smooth, continuous box-shadow scaling pulse that creates a modern, premium "breathing" light effect. It is completely lightweight and written in vanilla CSS.

## Tech Stack

- HTML
- CSS (no frameworks, no JavaScript)

## Preview

Open `demo.html` directly in your browser to see the effect.
