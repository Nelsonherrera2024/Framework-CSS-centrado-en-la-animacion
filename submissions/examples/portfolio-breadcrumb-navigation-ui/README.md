# Portfolio Breadcrumb Navigation UI

A breadcrumb navigation UI built using HTML and CSS to display the user's current location within a website.

---

## 🚀 Features

* Clear navigation path (Home → Portfolio → Projects)
* Active link highlighting
* Clean and minimal UI design
* Hover effects for better user experience
* Additional content section with cards

---

## 💡 Usage

Use this component in websites to show navigation hierarchy and improve user experience.

```html
<nav class="breadcrumb">
  <a href="#">Home</a>
  <span>/</span>
  <a href="#">Portfolio</a>
  <span>/</span>
  <a href="#" class="active">Projects</a>
</nav>
```

---

## 🧱 Structure

* Breadcrumb navigation bar
