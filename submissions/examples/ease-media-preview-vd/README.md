# Ease Media Preview

## What does this do?

A media preview component that displays an image or video thumbnail with a play button overlay, duration badge, and interactive hover animations.

---

## How is it used?

Open `demo.html` in any modern browser.

Example:

```html
<div class="media-preview">

    <div class="thumbnail">

        <img src="thumbnail.jpg" alt="Preview">

        <div class="play-overlay">▶</div>

        <div class="duration">
            03:42
        </div>

    </div>

</div>
```

Features:

- Image/video thumbnail
- Play button overlay
- Play button scales in on hover
- Duration badge
- Thumbnail zoom effect
- Responsive layout

---

## Why is it useful?

Media preview cards are commonly used in video platforms, streaming services, online courses, galleries, and content management systems. This component provides a clean, modern presentation with subtle hover animations, making it a great fit for EaseMotion CSS's animation-first design philosophy.
