# Modal Double Scrollbar Fix (Issue #14077)\n\n## What does this do?\nDemonstrates the corrected modal pattern where the container uses `overflow: hidden` and `display: flex; flex-direction: column`, with only `.ease-modal-body` set to `overflow-y: auto; flex: 1`.\n\n## How is it used?\n```html\n<div class="ease-modal">  <!-- overflow: hidden -->\n  <div class="ease-modal-body">  <!-- overflow-y: auto; flex: 1 -->\n    Long content...\n  </div>\n</div>\n```\n\n## Why is it useful?\nUsing `overflow: auto` on both the modal container and body creates two independent scroll contexts, showing two scrollbars. Setting `overflow: hidden` on the container and scrolling only via the body eliminates the double-scrollbar issue.
