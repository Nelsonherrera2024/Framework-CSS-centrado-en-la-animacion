# ease-loader-progress-dots

Three dots filling sequentially — classic typing/processing indicator.

## Usage

```html
<div class="ease-loader-progress-dots">
  <span></span><span></span><span></span>
</div>
```

## CSS Variables

| Variable | Default | Description |
|---|---|---|
| `--ease-loader-color` | `#6366f1` | Dot color |
| `--ease-loader-size` | `10px` | Size of each dot |
| `--ease-loader-speed` | `1.2s` | Animation duration |
| `--ease-loader-gap` | `6px` | Gap between dots |
