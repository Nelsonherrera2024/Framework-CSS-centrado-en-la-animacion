# White Space Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/white-space-utilities/style.css">
<div class="white-space-utilities-card white-space-utilities-card-featured">White Space Utilities Content</div>
```

## Classes
- `.white-space-utilities-card`: Primary card with breathe animation
- `.white-space-utilities-card-alt`: Secondary variant
- `.white-space-utilities-card-ghost`: Ghost/outline card
- `.white-space-utilities-card-glow`: Glow effect card
- `.white-space-utilities-card-featured`: Gradient featured card
- `.white-space-utilities-card-sm / -md / -lg`: Size variants
- `.white-space-utilities-v-1 through -5`: Opacity variants
- `.white-space-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.white-space-utilities-hover-lift / -scale / -glow`: Hover effects

## Design
- Hue: 271deg
- Border radius: 6px / 15px / 19px
