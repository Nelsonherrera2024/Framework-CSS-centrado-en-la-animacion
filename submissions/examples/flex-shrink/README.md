# Flex Shrink Utilities

CSS utility classes for the `flex-shrink` property.

## Usage

```html
<div class="shrink-0">...</div>
```

## Classes

- `.shrink-0` тАФ flex-shrink: 0;
- `.shrink` тАФ flex-shrink: 1;
- `.shrink-2` тАФ flex-shrink: 2;
- `.shrink-3` тАФ flex-shrink: 3;
- `.shrink-4` тАФ flex-shrink: 4;
- `.shrink-5` тАФ flex-shrink: 5;
- `.shrink-none` тАФ flex-shrink: 0;
- `.shrink-auto` тАФ flex-shrink: auto;
- `.shrink-inherit` тАФ flex-shrink: inherit;
- `.shrink-initial` тАФ flex-shrink: initial;
- `.shrink-unset` тАФ flex-shrink: unset;
- `.shrink-revert` тАФ flex-shrink: revert;
- `.shrink-safe` тАФ flex-grow: 1; flex-shrink: 1;
- `.shrink-unsafe` тАФ flex-grow: 0; flex-shrink: 1;
- `.shrink-equal` тАФ flex-grow: 1; flex-shrink: 1; flex-basis: 0;
- `.shrink-basis-auto` тАФ flex-shrink: 1; flex-basis: auto;
- `.shrink-basis-0` тАФ flex-shrink: 1; flex-basis: 0;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
