ease-break utilities

Control how elements break across columns and print pages.

ease-break-inside-avoid-za

Prevents the element from being split across columns or printed pages. The entire element stays together in one column.

ease-break-inside-auto-za

Default behavior — allows breaking inside the element if needed.

ease-break-before-za

Forces a column/page break before the element — it always starts at the top of a new column or page.

ease-break-before-avoid-za

Prevents a column/page break before the element — keeps it with the preceding content.

ease-break-after-za

Forces a column/page break after the element — next content starts in a new column or page.
