Bug: link transition not guarded

The a element transition on color is not suppressed under prefers-reduced-motion: reduce.

Hover this link — the color transition still plays.

Source

a {
  transition: color var(--ease-speed-fast) var(--ease-ease);
  /* no reduced-motion guard */
}

Fix

@media (prefers-reduced-motion: reduce) {
  a {
    transition: none !important;
  }
}
