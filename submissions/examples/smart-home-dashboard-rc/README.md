# Smart Home Control Dashboard Showcase

1. What does this do?

Provides an animated smart-home dashboard with device controls, automation management, energy monitoring, and activity tracking.

2. How is it used?

Open `demo.html` directly in a browser and explore the smart-home control interface.

3. Why is it useful?

It demonstrates practical IoT and smart-home UI patterns while showcasing multiple animation techniques in a realistic home automation dashboard.
