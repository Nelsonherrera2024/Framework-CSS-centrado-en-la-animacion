♿ prefers-reduced-motion Support

Respecting users' motion preferences for better accessibility

✅ How this works: If you have "Reduce Motion" enabled in your OS, animations will be disabled or reduced.

Test the Effect

These cards have animations. Enable/disable "Reduce Motion" in your OS settings to see the difference.

Bounce Animation
Pulse Animation
Rotate Animation
Fade Animation

How to Test

Windows: Settings → Accessibility → Visual Effects → Turn off "Animation Effects"

macOS: System Settings → Accessibility → Display → Reduce Motion

iOS: Settings → Accessibility → Motion → Reduce Motion

Android: Settings → Accessibility → Remove animations

Implementation Code

/* Add this to your CSS to respect reduced motion preferences */
@media (prefers-reduced-motion: reduce) {
    *,
    *::before,
    *::after {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
        transition-duration: 0.01ms !important;
        scroll-behavior: auto !important;
    }
}

/* Or for specific animations */
.ease-bounce {
    animation: bounce 1s ease infinite;
}

@media (prefers-reduced-motion: reduce) {
    .ease-bounce {
        animation: none;
    }
}
