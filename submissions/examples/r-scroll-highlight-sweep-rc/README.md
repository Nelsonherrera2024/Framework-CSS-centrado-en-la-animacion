# Scroll-Driven Text Highlight Sweep

## What does this do?

Demonstrates a viewport-driven text highlight animation that sweeps across important phrases as users scroll.

## Features

- Pure CSS implementation
- Uses `animation-timeline: view()`
- Scroll-linked highlight sweep
- Custom highlight colors
- Reduced-motion support
- Graceful fallback in unsupported browsers

## Usage

Open `demo.html` and scroll through the content to see highlighted phrases animate into view.

## Why is it useful?

This pattern is commonly used in premium landing pages, documentation sites, storytelling experiences, feature sections, and editorial content to draw attention to key phrases.
