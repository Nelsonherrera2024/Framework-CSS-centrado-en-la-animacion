# Animated Toggle Panel

Expandable FAQ-style accordion panels using the `<details>/<summary>` HTML element. Smooth height transition on open/close with rotating chevron indicator.

Created by @saurabhhhcodes

## Features
- Native `<details>/<summary>` semantics (accessible by default)
- Smooth open/close transition via `details[open]` selector
- Rotating chevron indicator using CSS transforms
- Multiple panels in FAQ-style layout
- BEM naming convention
