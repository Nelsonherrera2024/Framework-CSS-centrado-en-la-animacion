# Number Flip Counter

A digit flip counter with perspective animation.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--counter-bg` | `#2d2d44` | Counter background |
| `--digit-color` | `#4ecdc4` | Digit text color |
| `--flip-color` | `#3a5a5a` | Flip mid-state color |
| `--flip-duration` | `0.4s` | Flip animation speed |
