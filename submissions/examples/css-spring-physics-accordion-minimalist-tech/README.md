# CSS Spring Physics Accordion for Minimalist Tech Layouts

A pure CSS **Spring Physics Accordion** designed for **Minimalist Tech** interfaces. It uses native HTML `<details>` and `<summary>` elements to provide smooth spring-inspired expand/collapse animations without JavaScript.

## Features

- Pure HTML & CSS
- Spring physics expand/collapse animation
- Minimalist Tech design
- Responsive layout
- Keyboard accessible
- Uses semantic `<details>` and `<summary>`
- Customizable with CSS custom properties
- No JavaScript required

## Files

```
css-spring-physics-accordion-minimalist-tech/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` directly in any modern browser.
2. Customize colors, animation timing, border radius, and shadows through the CSS variables in `style.css`.

## Why it fits EaseMotion CSS

This component follows the EaseMotion CSS philosophy by providing a lightweight, reusable, animation-first accordion with smooth spring-inspired motion, responsive behavior, accessibility, and zero JavaScript dependency.
