# Animated Bookmark Save Flip

## Overview

A reusable bookmark icon animation featuring a subtle flip effect suitable for reading lists, documentation websites, blogs, educational platforms, and productivity applications.

## Features

- CSS-only animation
- Bookmark save flip effect
- Lightweight implementation
- Responsive demo
- Easy customization
- Accessibility support

## Usage

```html
<button class="bookmark-button">
  <span class="bookmark-icon">🔖</span>
</button>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
