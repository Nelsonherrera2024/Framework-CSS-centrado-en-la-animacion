# Text Decoration Style Utilities

CSS utility classes for the `text-decoration-style` property.

## Usage

```html
<div class="decoration-solid">...</div>
```

## Classes

- `.decoration-solid` — text-decoration-style: solid;
- `.decoration-double` — text-decoration-style: double;
- `.decoration-dotted` — text-decoration-style: dotted;
- `.decoration-dashed` — text-decoration-style: dashed;
- `.decoration-wavy` — text-decoration-style: wavy;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
