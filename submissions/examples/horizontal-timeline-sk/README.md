# Horizontal Timeline

A CSS-only horizontal timeline with active state and hover effects.

## Usage
```html
<div class="h-timeline">
  <div class="h-timeline__item h-timeline__item--active">
    <div class="h-timeline__dot"></div>
    <div class="h-timeline__content">
      <h3 class="h-timeline__title">Step</h3>
      <p class="h-timeline__desc">Description</p>
    </div>
    <div class="h-timeline__date">Date</div>
  </div>
</div>
```

## Why EaseMotion CSS?
Horizontal timeline variant with pseudo-element connector line, active indicator, and hover animations.
