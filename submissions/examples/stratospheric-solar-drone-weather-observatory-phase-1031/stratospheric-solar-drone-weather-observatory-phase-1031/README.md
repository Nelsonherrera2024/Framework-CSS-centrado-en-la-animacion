# Solar Horizons Weather Observatory Showcase

A full-page interactive UI demo built with **EaseMotion CSS**.  
This design demonstrates smooth 60fps animations, responsive layouts, and a futuristic aesthetic for stratospheric solar drone weather monitoring.

## 🧩 Components
- Hero with stratospheric imagery
- Features grid
- Mission timeline
- Drone visualization card
- Expert testimonials
- Call to action
- Contact form
- Footer

## 🎯 EaseMotion Classes Used
- `ease-fade` — fade entrances
- `ease-slide` — sliding headings
- `ease-scroll-reveal-scale` — reveal on scroll with scale
- `ease-hover-scale-icon` — hover scaling for CTAs and cards
- `ease-hover-opacity-reduce` — hover dimming for feature/review cards

## 🚀 Usage
1. Open `demo.html` directly in a browser.
2. Scroll to see entrance animations.
3. Hover over buttons/cards to see interactive effects.
4. Replace placeholder imagery with actual observatory visuals.

## 🧰 Technologies
- HTML5
- CSS3
- EaseMotion CSS framework
