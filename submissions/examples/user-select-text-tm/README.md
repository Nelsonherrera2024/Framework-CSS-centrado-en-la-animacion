`user-select-text-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/user-select-text-tm/style.css">
<div class="user-select-text-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .user-select-text: enables text selection

## Impact

Restores selection on elements with user-select-none
