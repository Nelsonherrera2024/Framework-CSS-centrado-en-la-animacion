# round-mod-math

Demonstrates CSS `round()`, `mod()`, and `rem()` math functions for numeric calculations with visual bar representations.

## How It Works

Three calculation panels show each function. `round()` snaps values to the nearest multiple, `mod()` returns the remainder (always non-negative), and `rem()` returns the remainder (sign matches dividend). Colored bars visualize the input, divisor, and result for each operation.

## Usage

Open `demo.html` in any modern browser. CSS custom properties control the test values and divisor. Toggle `prefers-reduced-motion` to disable bar animation. Each panel labels the calculation and result numerically.
