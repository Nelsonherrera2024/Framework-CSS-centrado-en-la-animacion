CSS round() mod() rem()

Calculations on numeric values with visual bar output

round()

Rounds value to the nearest multiple of step

round(17, 5) → 15
value (17)
step (5)
result (15)

mod()

Remainder (always non-negative)

mod(17, 5) → 2
value (17)
divisor (5)
result (2)

rem()

Remainder (sign matches dividend)

rem(-17, 5) → -2
value (-17)
divisor (5)
result (-2)
