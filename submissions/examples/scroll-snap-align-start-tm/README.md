`scroll-snap-align-start-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/scroll-snap-align-start-tm/style.css">
<div class="scroll-snap-align-start-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .scroll-snap-align-start: snaps to start edge

## Impact

Horizontal carousel and gallery snapping
