EaseMotion CSS Tooltips

A highly performant, accessible, pure-CSS tooltip system utilizing custom positions, entrance animations, and rich color modes.

1. Interactive Playground

Configure the options below to customize the tooltip in real-time. Hover or focus the preview button to see your configuration in action.

2. Positioning Variations

Tooltips can be positioned relative to their trigger element along the cardinal directions. Position flips automatically on mobile to fit the viewport.

3. Inline Text & Icon Triggers

Embed tooltips seamlessly on icons, inline text snippets, and badges. They are built using keyboard accessible focus-within events.

4. Semantic Colors & Branding

Color tooltips match their trigger action types, boosting usability and offering explicit cues for user operations.

5. Multi-line & Wide Layouts

Standard tooltips clip long paragraphs. Multi-line and Wide classes add paragraph formatting with automatic line wrapping.

API Quick Reference

Use standard combinations of HTML classes and attributes to configure tooltip behaviors:

Class / Attribute Role Syntax Example
.ease-tooltip Base component class wrapper <button class="ease-tooltip" ...>
data-tooltip="..." Defines the text inside the tooltip box data-tooltip="Hello world!"
.ease-tooltip-top/bottom/left/right Positions the tooltip block around the trigger class="ease-tooltip ease-tooltip-bottom"
.ease-tooltip-primary/success/... Color themes corresponding to feedback statuses class="ease-tooltip ease-tooltip-success"
.ease-tooltip-multiline/wide Enables multi-line wrapping and expanded text widths class="ease-tooltip ease-tooltip-wide"
