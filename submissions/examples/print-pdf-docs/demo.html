📄 Print PDF Docs

Generate a print-friendly PDF version of the full EaseMotion CSS documentation.

Overview

This tool builds a single PDF from the project's documentation files, making them easy to share, print, or read offline.

How It Works

  1. Detect available PDF engine (Playwright, WeasyPrint, or wkhtmltopdf)
  2. Read all documentation files from the docs/ directory
  3. Convert them into a single print-friendly PDF
  4. Output EaseMotion-CSS-Docs.pdf

Usage

bash build-pdf.sh

Make sure you have one of these installed:

  • Playwright — npm install playwright && npx playwright install chromium
  • WeasyPrint — pip install weasyprint
  • wkhtmltopdf — apt install wkhtmltopdf

Output

EaseMotion CSS Docs

EaseMotion-CSS-Docs.pdf
