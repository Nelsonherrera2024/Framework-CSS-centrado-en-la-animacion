# Animation: Feature Comparison Table

A responsive feature comparison table designed for SaaS products, developer tools, and documentation websites. The highlighted plan receives special emphasis while table cells respond to user interaction.

## Features

- Pure HTML & CSS
- Responsive comparison table
- Featured plan highlighting
- Interactive hover states
- Clean SaaS-inspired design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="comparison-table">

    <div class="table-header">
        Free
    </div>

    <div class="table-header featured">
        Pro
    </div>

</div>
```

## Why is it useful?

Comparison tables help users evaluate plans, features, and product offerings. This component provides a clear visual hierarchy and interactive experience while maintaining readability and accessibility.
