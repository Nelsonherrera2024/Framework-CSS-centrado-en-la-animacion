# Floating Label Input

A responsive input field with a floating label interaction built entirely with CSS.

## Features

- Pure CSS
- Accessible label association
- Responsive layout
- No JavaScript required

## Usage

```html
<div class="input-group">
  <input
    type="text"
    id="name"
    class="input-field"
    placeholder=" "
    required
  >
  <label for="name">Full Name</label>
</div>
```

The label automatically moves above the input when focused or when the field contains content.
