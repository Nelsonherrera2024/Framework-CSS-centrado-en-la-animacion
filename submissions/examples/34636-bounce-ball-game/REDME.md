# Neon Bounce Ball Game

A physics-based canvas game built for EaseMotion CSS. Players control a paddle to keep a ball bouncing within a dynamic, shrinking circular arena. 

## Features
- Custom physics engine with gravity and circular collision logic.
- Shrinking arena that increases difficulty over time.
- Power-up system integrated with leveling up (Multiball, Shields, Slow-mo).
- Responsive mouse and touch controls.
- Cyberpunk UI with canvas particle effects.

## How to Play
1. Open `demo.html` in any modern web browser.
2. Click **Start Game**.
3. Move your mouse (or swipe on touch screens) to move the paddle.
4. Keep the ball inside the glowing circle!
