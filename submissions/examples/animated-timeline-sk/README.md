# Animated Timeline

CSS-only vertical timeline with alternating milestones, connecting line, and hover effects.

## Features
- Vertical centered timeline with alternating left/right layout
- Connecting line with animated dot at each milestone
- Card-style content with arrow indicators
- Hover lift effect on milestone cards
- Dot scale animation on hover
- Fully responsive (stacks to single column on mobile)
- Clean, modern design

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`.

## Browser Compatibility
Modern browsers supporting CSS Flexbox, Transforms, and Custom Properties.
