1. Removed Unused Keyframe

@keyframes ease-kf-morph-card was defined at line 626 of core/animations.css but never referenced by any class in the framework. The .ease-hover-morph-card class uses CSS transitions instead of this keyframe.

Removed
@keyframes ease-kf-morph-card {
  from {
    border-radius: 8px;
    transform: scale(1);
  }
  to {
    border-radius: 50%;
    transform: scale(0.96);
  }
}

/* Removed — dead code,
   zero references in
   any CSS file */

2. Fixed Keyframe Name Mismatch

@keyframes ease-zoom-out (no -kf- infix) didn't match the reference ease-kf-zoom-out in .ease-zoom-out. This was a bug — the animation never ran.

Before (broken)
@keyframes ease-zoom-out {
  from { transform: scale(1.5); }
  to   { transform: scale(1); }
}

/* .ease-zoom-out references
   ease-kf-zoom-out — MISMATCH */
After (fixed)
@keyframes ease-kf-zoom-out {
  from { transform: scale(1.5); }
  to   { transform: scale(1); }
}

/* Names now match.
   Animation works correctly. */

3. Removed Redundant -webkit- Prefix

-webkit-scroll-snap-type is no longer needed — unprefixed scroll-snap-type has universal support since Safari 15 (2021).

Before
.ease-snap-x {
  scroll-snap-type: ...;
  -webkit-scroll-snap-type: ...; /* redundant */
}
After
.ease-snap-x {
  scroll-snap-type: ...;
  /* -webkit- prefix removed */
