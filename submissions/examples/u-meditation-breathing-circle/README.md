# ease-meditation-breathing-circle

A guided meditation breathing circle animation that expands and contracts to help users follow breathing exercises.

## Features

- Smooth breathing animation
- Expands and contracts slowly
- Dynamic breathing text cues
- Meditation themed UI
- Responsive design
- HTML, CSS and JavaScript
