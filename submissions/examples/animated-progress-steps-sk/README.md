# animated-progress-steps-sk

Horizontal progress steps wizard driven by radio buttons (no JavaScript). The progress bar fill animates smoothly between steps. Completed steps display a checkmark, the current step pulses with a ring animation, and upcoming steps appear dimmed/grey.

**Features:**
- Radio-button step selection
- Animated progress fill width
- Checkmark for completed steps
- Pulsing ring on current step
- Dimmed upcoming steps
