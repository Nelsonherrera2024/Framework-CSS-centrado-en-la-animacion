# Flex Basis Utilities

CSS utility classes for the `flex-basis` property.

## Usage

```html
<div class="basis-none">...</div>
```

## Classes

- `.basis-none` тАФ flex-basis: auto;
- `.basis-0` тАФ flex-basis: 0;
- `.basis-1\/2` тАФ flex-basis: 50%;
- `.basis-1\/3` тАФ flex-basis: 33.333%;
- `.basis-2\/3` тАФ flex-basis: 66.666%;
- `.basis-1\/4` тАФ flex-basis: 25%;
- `.basis-3\/4` тАФ flex-basis: 75%;
- `.basis-full` тАФ flex-basis: 100%;
- `.basis-px` тАФ flex-basis: 1px;
- `.basis-1rem` тАФ flex-basis: 1rem;
- `.basis-2rem` тАФ flex-basis: 2rem;
- `.basis-4rem` тАФ flex-basis: 4rem;
- `.basis-8rem` тАФ flex-basis: 8rem;
- `.basis-16rem` тАФ flex-basis: 16rem;
- `.basis-screen` тАФ flex-basis: 100vw;
- `.basis-min` тАФ flex-basis: min-content;
- `.basis-max` тАФ flex-basis: max-content;
- `.basis-fit` тАФ flex-basis: fit-content;
- `.basis-content` тАФ flex-basis: content;
- `.basis-auto` тАФ flex-basis: auto;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
