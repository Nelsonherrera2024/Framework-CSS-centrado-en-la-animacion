# Lightweight Documentation Search Bar

## What does this do?
This adds a real-time, lightweight search bar component intended for documentation layouts. It dynamically filters both sidebar navigation links and main-content table rows (like animation and utility references) on the client side as the user types.

## How is it used?
1. Type a query (like "fade" or "bounce") into the search input in the sidebar.
2. Watch the sidebar navigation items filter instantly.
3. Watch the data tables in the main content area update to only show rows matching the search term.

## Why is it useful?
As the number of utility classes and animations grows, finding a specific class becomes difficult for users navigating the documentation. A lightweight client-side search bar filters animations and examples in real time, drastically improving discoverability without requiring any backend search infrastructure.
