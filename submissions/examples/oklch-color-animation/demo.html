Transition quality assessment

OKLCH

Maintains consistent perceived lightness and saturation throughout the hue cycle. No "dead" spots — every intermediate color feels vibrant and balanced. The hue wheel transition is smooth because OKLCH is designed to match human perception of color differences.

Excellent

sRGB

Passes through desaturated, muddy gray regions between hues. Red to green transitions look brownish; green to blue can look cyan-washed. This happens because sRGB interpolates in a non-perceptual space where equal channel steps do not correspond to equal perceived differences.

Poor midpoints
/* OKLCH — smooth, perceptual hue cycling */
@keyframes oklchBgCycle {
  0%   { background: oklch(0.5 0.2 0); }
  25%  { background: oklch(0.6 0.25 90); }
  50%  { background: oklch(0.55 0.2 180); }
  75%  { background: oklch(0.6 0.25 270); }
  100% { background: oklch(0.5 0.2 360); }
}
