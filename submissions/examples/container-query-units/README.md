# Container Query Units

Explores CSS container query units (cqw, cqh, cqi, cqb, cqmin, cqmax) with resizable containers, live scaling cards, and a unit reference table.

## Features

- Side-by-side container panels demonstrating cqw, cqh, cqi, cqb, cqmin, cqmax
- Resizable card grid that scales font and spacing with cqi units
- Container queries that change grid layout at different container sizes
- Full unit reference table with viewport unit equivalents

## Usage

Open `demo.html` in a modern browser that supports container queries. Use the slider to resize the card container and observe how elements scale relative to container size instead of the viewport.
