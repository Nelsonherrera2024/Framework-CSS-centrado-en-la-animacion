# Knowledge Base Experience Center

An ultra-premium documentation and knowledge center featuring search-expanded inputs, sticky sidebar table-of-contents, category card lists, YAML syntax highlights, and feedback indicators. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Search Header**: Centered brand logo and input elements that expand on focus.
- **Hero Area**: Help statement with list of trending links.
- **Category Matrix**: Four card grids detailing getting started steps, API references, tutorials, and FAQs.
- **Sidebar TOC Layout**: Navigation links locked via CSS sticky boundaries.
- **Content Area**: Doc details displaying code blocks, headers, paragraphs, and rating badges.
- **Feedback Widget**: Helpful review query buttons with hover triggers.
- **Recently Updated List**: File links displaying status tags and update timings.

## 🎨 Theme Details

- **Palette**: Dark Charcoal (`#111827`), Slate Grey (`#1f2937`), Sky Blue (`#0ea5e9`), Success Green (`#10b981`), Violet Purple (`#8b5cf6`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Search expands, entrance fade-in-ups, code highlights, and hover lifts.
