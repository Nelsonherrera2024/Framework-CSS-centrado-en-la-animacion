# Scroll Text Reveal

## Overview
This feature introduces a scroll-based text reveal animation where content appears line-by-line with a staggered delay.

## ✨ Features
- Scroll-triggered animation
- Staggered text reveal
- Smooth transitions
- Lightweight and reusable

## 🎯 Utility Classes
- `reveal`
- `ease-fade-in`
- `ease-hover-grow`

## 💡 Usage

```html
<p class="reveal">Your text here</p>
