# CSS File Tree

A file explorer tree view using `<details>`/`<summary>` elements and pure CSS.

## Usage
```html
<div class="file-tree">
  <details class="file-tree__folder">
    <summary class="file-tree__label">src/</summary>
    <div class="file-tree__file">index.js</div>
  </details>
</div>
```

## Why EaseMotion CSS?
Reusable folder/file tree navigation pattern without JavaScript.
