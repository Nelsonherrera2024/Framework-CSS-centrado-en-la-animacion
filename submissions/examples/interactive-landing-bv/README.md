# Interactive Landing Page System

An ultra-premium, conversion-optimized landing page built with pure HTML and EaseMotion CSS. Displays hero actions, problem/solution grids, testimonials, timelines, pricing plans, and cookie banners.

---

## ✨ Features

### 18 Unique Sections

- **Sticky Navigation**: Transparent-to-solid transitions as you scroll.
- **Hero Area**: Linear-gradient headline texts with CTA button elements.
- **Social Proof Bar**: Trusted partners alignment.
- **Problem vs Solution**: Old vs New system design comparison cards.
- **Feature Matrix**: Observability, budget automation, and SLA telemetry cards.
- **Code Tab**: Interactive mockup card.
- **Numeric Counter**: Observability metrics row.
- **Setup Stepper**: Setup pipeline steppers showing deployment steps.
- **Pricing Cards**: Three options (Sandbox, Pro, Enterprise) with check features.
- **Testimonial Slider**: Detailed feedback block.
- **Ecosystem Integration**: Vendors logo block.
- **FAQ Accordion**: Expandable FAQ details.
- **Newsletter Signup**: Dynamic email form inputs.
- **Blog Preview**: Resource article outlines.
- **CTA Banner**: Bottom callout section with linear gradients.
- **Site Footer**: Copy details and page navigation links.
- **Scroll to Top**: Floating button returning users to top of viewport.
- **Cookie Consent**: Bottom banner explaining privacy.

## 🎨 Theme & Typography

- **Palette**: Dark Blue BG (`#030712`), Blue Primary (`#2563eb`), Secondary Purple (`#7c3aed`), Warning Orange (`#ea580c`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft entrances, card hover scaling, and transition buttons.
