CSS interpolate-size

Smooth height animation between fixed and intrinsic values height: auto — no JavaScript!

What is interpolate-size?

CSS transitions have never been able to animate height: auto — until now. interpolate-size: allow-keywords lets you smoothly transition between any length and intrinsic keyword (auto, min-content, max-content, fit-content).

Browser Support for interpolate-size

As of 2026, interpolate-size is supported in Chrome 133+, Edge 133+, and Safari 18.4+. Firefox support is in development. Check caniuse.com for the latest status.

The property is safe to use as a progressive enhancement — without it, the content still shows/hides instantly.

Usage Tips
  • Set interpolate-size: allow-keywords on :root for global use
  • Apply transition: height 0.3s ease or similar
  • Works with grid-template-rows for expandable grids
  • Combine with @starting-style for entry animations
How It Works

Previously, height: auto was a "discrete" value that couldn't be interpolated. interpolate-size: allow-keywords changes this, allowing the browser to calculate the auto height and animate between it and the specified height.

The details element is a natural fit — its height changes when the open attribute toggles. With interpolate-size, this transition becomes smooth.
