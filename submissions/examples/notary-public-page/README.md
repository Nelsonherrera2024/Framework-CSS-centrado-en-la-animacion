# Professional Services — Notary Public

## What does it do?
A complete, copy-paste ready full-page website for a Notary Public
professional services business. Built using only EaseMotion CSS classes.

## EaseMotion Classes Showcased
- `.ease-fade-in` — hero text and section entrances
- `.ease-slide-up` — cards and content blocks
- `.ease-slide-in-left` / `.ease-slide-in-right` — split sections
- `.ease-zoom-in` — pricing cards
- `.ease-hover-grow` — service cards, buttons
- `.ease-hover-lift` — pricing cards
- `.ease-hover-glow` — CTA buttons
- `.ease-hover-shimmer` — feature highlights
- `.ease-card-lift` — review cards
- `.ease-pulse` — availability badge
- `.ease-delay-100` through `.ease-delay-500` — stagger effects
- `.ease-duration-slow` — hero animations
