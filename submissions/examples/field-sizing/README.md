# CSS field-sizing

## Overview
This example demonstrates the CSS ield-sizing: content property that makes form controls (textarea, input, select) automatically resize based on their content. No more JavaScript for auto-growing textareas!

## Features
- **ield-sizing: content** — makes form elements size to fit their content
- **ield-sizing: fixed** — default behavior, elements have a fixed size
- **Auto-growing textareas** — grow vertically as user types
- **Input sizing** — inputs expand horizontally with text length
- **Min/max constraints** — works with standard min-width, max-width, etc.

## How to Use
1. Add ield-sizing: content to textareas, inputs, or selects
2. Set min-height / max-height to constrain growth for textareas
3. Set min-width / max-width to constrain growth for inputs
