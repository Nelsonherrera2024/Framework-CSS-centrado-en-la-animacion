# Will Change Utilities

CSS utility classes for the `will-change` property.

## Usage

```html
<div class="will-change-auto">...</div>
```

## Classes

- `.will-change-auto` тАФ will-change: auto;
- `.will-change-scroll` тАФ will-change: scroll-position;
- `.will-change-contents` тАФ will-change: contents;
- `.will-change-transform` тАФ will-change: transform;
- `.will-change-opacity` тАФ will-change: opacity;
- `.will-change-top-left` тАФ will-change: top, left;
- `.will-change-cursor` тАФ will-change: cursor;
- `.will-change-font` тАФ will-change: font-variation-settings;
- `.will-change-scrollx` тАФ will-change: scroll-position;
- `.will-change-auto-2` тАФ will-change: auto;
- `.will-change-multi` тАФ will-change: transform, opacity;
- `.will-change-multi-2` тАФ will-change: transform, opacity, contents;
- `.will-change-inherit` тАФ will-change: inherit;
- `.will-change-initial` тАФ will-change: initial;
- `.will-change-unset` тАФ will-change: unset;
- `.will-change-default` тАФ will-change: auto;
- `.change-auto` тАФ will-change: auto;
- `.change-transform` тАФ will-change: transform;
- `.change-opacity` тАФ will-change: opacity;
- `.change-scroll` тАФ will-change: scroll-position;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
