# CSS-only Cyberpunk Glitch Button

A futuristic cyberpunk-inspired button built entirely with CSS.

## Features

- Neon cyberpunk styling
- CSS-only glitch animation
- Scanline effect
- Geometric clip-path shape
- Hover glow interaction
- No JavaScript required

## Files

- demo.html
- style.css

## Usage

Add the `cyber-btn` class to any button element.

```html
<button class="cyber-btn">
<span class="glitch-text" data-text="LAUNCH">LAUNCH</span></button>
```

## Why it fits EaseMotion CSS

This component showcases animation-first design using pure CSS effects, hover interactions, and visually engaging motion while remaining lightweight and reusable.
