# pixel feedback score card

Adds a feedback score card with animated rating bars, customer signal copy, and responsive summary layout.

## What is included

- demo.html contains the standalone example structure.
- style.css includes the responsive layout, animation states, and reduced motion handling.
- The component keeps status text visible while using motion as a supporting cue.

## Usage

Open demo.html directly or copy the example folder into an EaseMotion showcase page.
