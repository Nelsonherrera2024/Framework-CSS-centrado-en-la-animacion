# EduLift – Online Course Platform Landing Page

A fully responsive, animated landing page for an EdTech SaaS product, built entirely with **EaseMotion CSS** utility classes.

## Sections
- Hero with course preview carousel
- Categories navigation
- Featured courses grid (rating, price)
- Instructor spotlight
- Student testimonials
- Pricing/subscription tiers
- Newsletter signup

## EaseMotion Classes Showcased
- **Layout:** ease-grid, ease-grid-cols-*, ease-container, ease-flex
- **Typography:** ease-text-*, ease-font-*
- **Colours & Backgrounds:** ease-bg-*, ease-text-*
- **Components:** ease-card, ease-btn, ease-badge, ease-input
- **Hover Effects:** ease-hover-scale-105, ease-hover-shadow-xl, ease-hover-rotate-1
- **Animations:** ease-fade-in, ease-slide-up, ease-delay-*
- **Spacing & Sizing:** ease-p-*, ease-m-*, ease-gap-*
- **Responsive:** ease-md-grid-cols-*, ease-md-w-auto

## How to Use
1. Copy demo.html into your project.
2. Ensure the path to easemotion.css is correct (usually from core/).
3. Open demo.html in any modern browser – no build step required.
