# Forensic Accountant Template
A serious, analytical, and highly professional template for forensic accounting firms, fraud investigation services, and litigation support consultancies.

- **Design**: Uses an "Investigative-Accuracy" palette with investigation-slate grays, accuracy-blue highlights, and a clean, evidence-focused layout to communicate diligence, neutrality, and expert detail.
- **Use Case**: Forensic audit firms, fraud detection consultancies, and legal support experts.
