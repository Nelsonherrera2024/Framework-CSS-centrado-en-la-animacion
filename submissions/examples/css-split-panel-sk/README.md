# CSS Split Panel

Resizable split panel layout using CSS flexbox and `:has()`.

## Usage
```html
<div class="split-panel">
  <div class="split-panel__pane split-panel__pane--left">Left</div>
  <div class="split-panel__divider"></div>
  <div class="split-panel__pane split-panel__pane--right">Right</div>
</div>
```

## Why EaseMotion CSS?
Demonstrates CSS layout patterns with interactive resizing.
