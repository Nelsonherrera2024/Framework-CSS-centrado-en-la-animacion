# ease-text-reveal-char

Splits text into individual `<span>` elements per character and staggers their entrance with a spring-like transition.

**Usage:** Include `style.css` and the inline script from `demo.html`. Target any text element with `id="revealText"`. Each character animates from `translateY(20px)` to `translateY(0)` with a 30ms staggered delay.
