# EaseMotion CSS — Breadcrumb Navigation Showcase

A premium, interactive showcase displaying the **EaseMotion CSS** Breadcrumb navigation component.

## Showcase Features

*   **Breadcrumb List Structure**: Demonstrates structured `ol > li` elements using standard framework navigation lists.
*   **Flexible Typography Sizing**: Previews `.ease-breadcrumb-sm`, standard size, and `.ease-breadcrumb-lg` classes.
*   **Custom Separator Configurator**: Demonstrates changing the active separators (slashes, arrows, double-chevrons, bullet dots) using an interactive configurations utility.
*   **Link Hover Underline Transitions**: Custom-designed clean animated underline transitions built using native framework speeds.
*   **Mobile Collapsible Layout**: Auto-collapses long paths into an ellipsis container for mobile screen dimensions to prevent layout truncation errors.

---

## EaseMotion CSS Classes Showcased

### Layout & Spacing
*   `.ease-flex` — Standard display flex.
*   `.ease-flex-wrap` — Wraps content.
*   `.ease-justify-between` — Aligns navigation content headers.
*   `.ease-items-center` — Vertically aligns content items.
*   `.ease-gap-2` — Horizontal elements gap.
*   `.ease-stack-lg` — Vertical layout spacing.

### Components

#### Breadcrumbs
*   `.ease-breadcrumb` — Core breadcrumb navigation container.
*   `.ease-breadcrumb-list` — Flex list layout block with custom overrides.
*   `.ease-breadcrumb-item` — Layout items container.
*   `.ease-breadcrumb-link` — Anchor styling with primary brand color transitions.
*   `.ease-breadcrumb-separator` — Separator symbol spacing control.
*   `.ease-breadcrumb-active` — Non-interactive text styling highlighting the current active page.
*   `.ease-breadcrumb-sm` / `.ease-breadcrumb-lg` — Sizing modifier classes.

#### Cards
*   `.ease-card` — Standard card layout container.
*   `.ease-card-outlined` — Outlined border variant.
*   `.ease-card-header` / `.ease-card-body` — Cards content organization sections.
*   `.ease-card-flat` — Container background panel.

#### Buttons & Badges
*   `.ease-btn` — Base button style properties.
*   `.ease-btn-outline` — Border-stroke styling.
*   `.ease-btn-sm` — Small button size variant.
*   `.ease-btn-hover` — Interactive scale-lift hover trigger.
*   `.ease-badge` — Status indicators.

### Animations
*   `.ease-fade-in` — Entrance loading animation.
