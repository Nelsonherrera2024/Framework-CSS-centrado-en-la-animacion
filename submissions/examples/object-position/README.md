# Object Position Utilities

CSS utility classes for the `object-position` property.

## Usage

```html
<div class="object-top">...</div>
```

## Classes

- `.object-top` — object-position: top;
- `.object-bottom` — object-position: bottom;
- `.object-left` — object-position: left;
- `.object-right` — object-position: right;
- `.object-center` — object-position: center;
- `.object-left-top` — object-position: left top;
- `.object-left-bottom` — object-position: left bottom;
- `.object-right-top` — object-position: right top;
- `.object-right-bottom` — object-position: right bottom;
- `.object-top-center` — object-position: top center;
- `.object-bottom-center` — object-position: bottom center;
- `.object-center-top` — object-position: center top;
- `.object-center-bottom` — object-position: center bottom;
- `.object-50-50` — object-position: 50% 50%;
- `.object-0-0` — object-position: 0 0;
- `.object-100-0` — object-position: 100% 0;
- `.object-0-100` — object-position: 0 100%;
- `.object-100-100` — object-position: 100% 100%;
- `.object-25-25` — object-position: 25% 25%;
- `.object-75-75` — object-position: 75% 75%;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
