# Ease WebSocket Live Connection

## Overview

Ease WebSocket Live Connection is a reusable visualization component that demonstrates a persistent real-time connection between a client and server using animated data packets and connection status indicators.

---

## Features

- 🔄 Live WebSocket visualization
- 💻 Client node
- ☁️ Server node
- 🟢 Animated data packets
- 💚 Pulsing connection status
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Client endpoint
- Server endpoint
- Animated connection wire
- Moving data packet
- Live status indicator

---
