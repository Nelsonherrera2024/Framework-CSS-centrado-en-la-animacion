Overscroll Behavior

Control Scroll Chaining and Bounce

The overscroll-behavior property controls what happens when a scroll boundary is reached. Prevents scroll chaining in nested containers and stops unwanted browser pull-to-refresh.

Pull-to-Refresh Suppression
On mobile, overscroll-behavior: contain on the body prevents the pull-to-refresh gesture from triggering on non-refreshable pages.
overscroll-behavior: contain on body

This container captures all scroll events. Scrolling to the top or bottom stops here — the browser chrome will not pull down or up. Useful for embedded feeds, chat interfaces, and game canvases.

EaseMotion CSS utilities use !important to ensure overscroll behavior applies consistently across all nested scroll containers. Dark mode and reduced-motion are fully supported.
