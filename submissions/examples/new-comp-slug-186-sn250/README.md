# Premium Glassmorphism Pricing Card

## 🎨 Component Overview
A premium, CSS-only pricing card component featuring glassmorphism effects, gradient animations, and smooth hover interactions. Perfect for SaaS products, subscription plans, or premium feature showcases.

## ✨ Features
- **Pure CSS** - Zero JavaScript dependencies
- **Glassmorphism Design** - Modern frosted glass aesthetic
- **Gradient Animations** - Smooth animated border gradients
- **Responsive** - Adapts to all screen sizes
- **Interactive** - Hover effects with scale and glow
- **Accessibility** - Reduced motion support
