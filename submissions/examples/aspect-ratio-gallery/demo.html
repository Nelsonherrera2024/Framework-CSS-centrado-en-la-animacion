Aspect Ratio vs. Height: Auto

aspect-ratio: 16/9

img { width: 100%; aspect-ratio: 16 / 9; }

Space is reserved before the image loads — no layout shift.

height: auto (old way)

img { width: 100%; height: auto; }

Zero height until the image loads — causes CLS.

/* Recommended pattern for responsive media */
img, video {
  width: 100%;
  aspect-ratio: attr(width) / attr(height);
  object-fit: cover;
}
