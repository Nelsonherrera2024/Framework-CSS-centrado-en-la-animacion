# Special Education Resource Center

A complete, responsive landing page for an **Inclusive Special Education Resource Center**, built as an EaseMotion CSS example. The page is designed to showcase how existing `ease-*` animation and interaction classes can be combined to create a modern, accessible education website without requiring a build step.

## Overview

The page demonstrates a welcoming, inclusive design for educators, parents, specialists, and families seeking special education resources.

It includes realistic placeholder content and highlights common sections found on educational resource websites.

## Included Sections

* Hero section
* Resources by Learning Difference
* Programs for Teachers
* Programs for Parents
* Specialist Profiles
* Family Testimonials
* Free Resource Downloads
* Newsletter Signup
* Footer

## EaseMotion CSS Classes Showcased

This example uses existing EaseMotion CSS utility classes, including:

* `ease-fade-up`
* `ease-scale-in`
* `ease-slide-left`
* `ease-slide-right`
* `ease-hover-lift`

No custom animation keyframes are introduced. Animations rely on the existing EaseMotion CSS utilities.

## Usage

1. Clone the EaseMotion CSS repository.
2. Navigate to:

```text
submissions/examples/u-special-education-resource-center-21041/
```

3. Open `demo.html` directly in your browser.

Ensure the EaseMotion CSS stylesheet is correctly linked so the `ease-*` utility classes are available.
