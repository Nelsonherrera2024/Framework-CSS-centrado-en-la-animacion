@view-transition Rule

The @view-transition at-rule enables navigation-based transitions. Use navigation: auto; to opt in.

view-transition-name

Assign matching names across pages so elements morph rather than fade. Each named element gets its own pseudo-element tree.

Browser Support

Chrome 111+, Edge 111+. Firefox and Safari are actively developing support.
