# Animated Radio Group

CSS-only custom radio buttons with selection animations, multiple orientations, and card-style variants.

## Features
- Horizontal and vertical orientations
- Custom styled radio circles with fill animation
- Card-style variant with icons
- Focus-visible accessibility
- Smooth selection transitions
- Price labels support for pricing plans
- Responsive layout

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`.

## Browser Compatibility
Modern browsers supporting CSS Flexbox, Transforms, and :checked pseudo-class.
