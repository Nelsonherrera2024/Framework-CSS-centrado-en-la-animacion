2. Mixins Configuration Guide

Here is how the mixins are structured and included inside custom stylesheets under the namespaced module system:

Exposed Mixins Structure (scss/_mixins.scss) 
// Namespaced import of variables module
@use 'variables';

@mixin animate($name, $duration: variables.$duration-normal, ...) {
  animation-name:            $name;
  animation-duration:        $duration;
  ...
}

@mixin slide-up($duration: variables.$duration-normal, ...) {
  @include animate(ease-kf-slide-up, $duration, ...);
}
Importing and Including Mixins (style.scss) 
// Load mixins cleanly as a namespaced package
@use '../../../scss/mixins';

.active {
  .slide-box {
    // Invoke mixin using namespaced prefix
    @include mixins.slide-up($duration: 800ms);
  }
}
