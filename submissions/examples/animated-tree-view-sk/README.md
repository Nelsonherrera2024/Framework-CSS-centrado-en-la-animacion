# Animated Tree View

A CSS-only tree view / nested list component using the checkbox hack for expand/collapse. Features connecting lines and folder/file icons.

## Created By

[@saurabhhhcodes](https://github.com/saurabhhhcodes)

## Features

- **Nested indented levels** with visual hierarchy
- **Expand/collapse** via hidden checkbox (no JS required)
- **Connecting lines** from parent to children
- **Folder / file icons** using pure CSS (Unicode/box-drawing characters)

## Usage

1. Link `style.css` in your HTML.
2. Build nested `<ul>` / `<li>` structure following BEM classes.
3. Toggle open/closed by clicking the label.
