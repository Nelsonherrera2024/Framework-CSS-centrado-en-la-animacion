# Forced Color Adjust Utilities

CSS utility classes for controlling color adjustment in forced-colors mode (Windows High Contrast).

## Classes

- `.forced-color-adjust-auto` — `forced-color-adjust: auto`
- `.forced-color-adjust-none` — `forced-color-adjust: none`

## Why

Prevents browser color overrides in high-contrast mode, preserving custom button, badge, or chart colors when accessibility features are active.
