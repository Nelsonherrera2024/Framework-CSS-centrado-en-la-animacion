# Animation Name Utilities

CSS utility classes for the `animation-name` property.

## Usage

```html
<div class="anim-none">...</div>
```

## Classes

- `.anim-none` — animation-name: none;
- `.anim-fade-in` — animation-name: fadeIn;
- `.anim-fade-out` — animation-name: fadeOut;
- `.anim-slide-up` — animation-name: slideUp;
- `.anim-slide-down` — animation-name: slideDown;
- `.anim-slide-left` — animation-name: slideLeft;
- `.anim-slide-right` — animation-name: slideRight;
- `.anim-bounce-in` — animation-name: bounceIn;
- `.anim-bounce-out` — animation-name: bounceOut;
- `.anim-spin` — animation-name: spin;
- `.anim-ping` — animation-name: ping;
- `.anim-pulse` — animation-name: pulse;
- `.anim-pulse-grow` — animation-name: pulseGrow;
- `.anim-shake` — animation-name: shake;
- `.anim-swing` — animation-name: swing;
- `.anim-tada` — animation-name: tada;
- `.anim-wiggle` — animation-name: wiggle;
- `.anim-jello` — animation-name: jello;
- `.anim-heartbeat` — animation-name: heartbeat;
- `.anim-flash` — animation-name: flash;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
