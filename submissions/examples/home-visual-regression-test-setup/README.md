# 🚀 Visual Regression Test UI

A simple UI layout designed specifically for visual regression testing.

---

## ✨ Features

- Clean and consistent UI  
- Multiple components for comparison  
- Hover interaction for visual testing  
- Stable layout for snapshot testing  

---

## 📦 Use Cases

- Visual regression testing  
- UI consistency checks  
- Screenshot comparison tools (Playwright)  
- Testing UI changes over time  

---

## 🧩 How It Works

- This UI is used as a test target  
- Tools like Playwright capture screenshots  
- Snapshots are compared to detect changes  

---

## 🛠️ Tech Used

- HTML  
- CSS  

---

## 💡 Why this fits EaseMotion CSS

This ensures UI consistency and helps detect unwanted design changes, improving overall UI quality.
