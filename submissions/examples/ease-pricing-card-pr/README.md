# Pricing Card Component

## What does this do?

Provides a modern pricing card component with featured plan highlighting, hover animations, and responsive layouts.

## How is it used?

```html
<div class="pricing-card featured">
  <div class="badge">Most Popular</div>

  <h3>Pro Plan</h3>

  <div class="price">
    <span>$19</span>/month
  </div>

  <button>Get Started</button>
</div>
```

## Why is it useful?

Pricing sections are a common requirement for SaaS platforms, subscription services, product landing pages, and startup websites. This component offers a clean and reusable pricing card with animation-first interactions that align with EaseMotion CSS's design philosophy.
