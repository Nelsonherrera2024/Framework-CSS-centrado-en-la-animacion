# Sub-Orbital Space Tourism Booking Portal

## Overview

This example is a premium futuristic booking portal for a sub-orbital travel experience. It is built as a direct-open static showcase using only HTML and CSS, with all visuals created through inline SVG and layered CSS effects.

## Features

- Luxury aerospace visual direction with high-contrast dark surfaces
- Hero panel with central rocket, capsule, launch pad, stars, orbital path, and Earth horizon
- Four floating status badges for launch and cabin readiness
- Flight package cards for three curated voyage tiers
- Booking timeline covering reservation through liftoff
- Passenger readiness dashboard with animated progress bars
- Orbit altitude and launch window intelligence cards
- Spaceport destination cards for multiple departure environments
- Safety and training checklist using native form controls
- Luxury cabin experience section with comfort metrics
- CTA booking controls with static reservation inputs
- Responsive layout for desktop, tablet, and mobile
- No JavaScript and no external assets

## EaseMotion Classes Used

No EaseMotion framework classes are used in the page markup. Custom scoped classes were used instead to satisfy the no-framework requirement while keeping the demo fully self-contained.

## How To Open demo.html

Open `demo.html` directly in any modern browser. No build step, local server, or dependency install is required.
