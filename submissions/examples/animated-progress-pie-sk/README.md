# Animated Progress Pie

Circular pie chart progress indicator using `conic-gradient`. Percentage value centered in the middle. Multiple examples (25%, 50%, 75%, 100%) with color that changes based on threshold.

Created by @saurabhhhcodes

## Features
- Pure CSS conic-gradient for the pie fill
- Percentage label centered via flexbox
- Dynamic color: red (<50%), amber (<80%), green (>=80%)
- 4 demo variants: 25%, 50%, 75%, 100%
- BEM naming convention
