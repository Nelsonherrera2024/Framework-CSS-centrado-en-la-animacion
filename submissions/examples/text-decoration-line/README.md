# Text Decoration Line Utilities

CSS utility classes for the `text-decoration-line` property.

## Usage

```html
<div class="underline">...</div>
```

## Classes

- `.underline` ‚Äî text-decoration-line: underline;
- `.overline` ‚Äî text-decoration-line: overline;
- `.line-through` ‚Äî text-decoration-line: line-through;
- `.no-underline` ‚Äî text-decoration-line: none;
- `.blink` ‚Äî text-decoration-line: blink;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
