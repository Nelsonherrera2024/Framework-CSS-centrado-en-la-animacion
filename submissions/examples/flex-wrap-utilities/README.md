# Flex Wrap Utilities

## Overview
Card utility classes for the EaseMotion CSS framework.

## Usage
```html
<link rel="stylesheet" href="submissions/examples/flex-wrap-utilities/style.css">
<div class="flex-wrap-utilities-card flex-wrap-utilities-card-featured">Flex Wrap Utilities Content</div>
```

## Classes
- `.flex-wrap-utilities-card`: Primary card with breathe animation
- `.flex-wrap-utilities-card-alt`: Secondary variant
- `.flex-wrap-utilities-card-ghost`: Ghost/outline card
- `.flex-wrap-utilities-card-glow`: Glow effect card
- `.flex-wrap-utilities-card-featured`: Gradient featured card
- `.flex-wrap-utilities-card-sm / -md / -lg`: Size variants
- `.flex-wrap-utilities-v-1 through -5`: Opacity variants
- `.flex-wrap-utilities-shd-xs / -sm / -md / -lg / -xl`: Shadow variants
- `.flex-wrap-utilities-hover-lift / -scale / -glow`: Hover effects
