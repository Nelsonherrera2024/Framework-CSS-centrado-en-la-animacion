# Pointer Events Utilities

CSS utility classes for the `pointer-events` property.

## Usage

```html
<div class="pointer-none">...</div>
```

## Classes

- `.pointer-none` — pointer-events: none;
- `.pointer-auto` — pointer-events: auto;
- `.pointer-stroke` — pointer-events: stroke;
- `.pointer-fill` — pointer-events: fill;
- `.pointer-all` — pointer-events: all;
- `.pointer-visible` — pointer-events: visible;
- `.pointer-visibility` — pointer-events: visiblePainted;

## Responsive

- `sm:` prefix for 640px+
- `lg:` prefix for 1024px+

## Dark Mode

- `dark:` prefix for `prefers-color-scheme: dark`
## Reduced Motion

- `motion-safe:` prefix for `prefers-reduced-motion: reduce`
