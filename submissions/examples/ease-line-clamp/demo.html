Text Truncation & Line-Clamp

New utility classes for handling long text overflow gracefully.

Single Line Truncation

.ease-truncate

This is a very long sentence that will be truncated with an ellipsis because it exceeds the width of its container.

💡 .ease-truncate uses text-overflow: ellipsis and prevents wrapping.

Multi-Line Clamp

.ease-line-clamp-2

EaseMotion CSS is a modern, animation-first utility library. This paragraph is clamped to exactly two lines. If there is more text, it will be hidden and replaced with an ellipsis at the end of the second line. This is extremely useful for card descriptions, news previews, and list items where vertical space is at a premium.

.ease-line-clamp-3

The line-clamp utility uses the -webkit-line-clamp property, which is widely supported across all modern browsers and has now been standardized in CSS. It provides a clean way to limit text height without needing complex JavaScript or overflow: hidden hacks that cut off text in the middle of a line. By clamping to 3 lines, we ensure a consistent layout while still giving users enough context.
