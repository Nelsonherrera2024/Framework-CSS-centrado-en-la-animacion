# Statistics Dashboard Cards

A modern, responsive collection of statistics dashboard cards built using pure HTML and CSS.

This example demonstrates reusable KPI cards featuring trend badges, progress indicators, hover animations, responsive layouts, dark mode support, and accessibility best practices.

---

## Features

- Responsive CSS Grid layout
- Modern KPI/statistics cards
- Positive and negative trend badges
- Animated progress bars
- Card hover lift effect
- Icon hover animation
- Dark mode support (`prefers-color-scheme`)
- Reduced motion support (`prefers-reduced-motion`)
- Focus-visible accessibility
- Semantic HTML structure
- Pure HTML and CSS (no JavaScript)
