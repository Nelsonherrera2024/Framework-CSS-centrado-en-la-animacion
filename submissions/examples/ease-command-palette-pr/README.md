# Command Palette

## What does this do?

Provides a modern command palette component with searchable actions and keyboard shortcut indicators.

## How is it used?

```html
<div class="palette">
  <div class="search-box">
    <input type="text" placeholder="Search commands..." />
  </div>
</div>
```

## Why is it useful?

Command palettes are commonly used in dashboards, developer tools, productivity applications, and SaaS products to help users quickly navigate and execute actions through a centralized interface.
