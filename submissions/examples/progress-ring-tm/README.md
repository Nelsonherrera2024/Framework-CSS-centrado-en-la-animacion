# Progress Ring

## What does this do?
Adds `.ease-progress-ring` — a circular progress indicator using SVG.

```html
<svg class="ease-progress-ring" style="--ring-value: 75%">
  <circle class="ease-ring-track"/>
  <circle class="ease-ring-fill"/>
</svg>
```

## Preview
Open `demo.html` in your browser.
