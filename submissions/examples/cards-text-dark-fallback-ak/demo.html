Cards Text Dark Fallback Bug

Hardcoded light-theme text colors in cards break dark mode readability.

Issue #5945

The Problem

Card components use hardcoded color: #1f2937 (dark text) suitable only for light backgrounds. In dark mode, the card background changes to dark while the text stays dark — text becomes invisible.
