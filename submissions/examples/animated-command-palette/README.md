# Animated Command Palette

A modern command palette component inspired by VS Code, Linear, and Notion.

## Features

- Animated command palette modal
- Search input field
- Command list UI
- Keyboard shortcut badges
- Hover animations
- Smooth modal appearance
- Responsive layout
- Pure HTML and CSS

## Folder Structure

```text
animated-command-palette/
├── demo.html
├── style.css
└── README.md
```

## Included Commands

### Navigation

- Home
- Projects
- Dashboard

### Actions

- Create New File
- Settings
- Search Workspace

## Animations

- Modal open animation
- Command hover transitions
- Keyboard shortcut highlight
- Active command state

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Responsive Design

## Usage

1. Open demo.html in a browser.
2. Ensure style.css is in the same folder.
3. Customize commands and shortcuts as required.
