# Global Monitoring Grid

An enterprise-grade, worldwide telemetry observer and global operations command dashboard featuring edge node capacity statistics, operational incident registries, load balance rerouting controls, and active telemetry logs. Built with pure HTML and EaseMotion CSS.

---

## ✨ Features

### Key UI Sections

- **Grid Observer Header**: Command logos, global active states, and platform security ratings.
- **Worldwide Telemetry Cards**: Quad layouts detailing average SLA uptimes, active edge nodes counts, latency speeds, and compliance postures.
- **Incident Registry Table**: Tables detailing incident codes, region locations, impact parameters, load metrics, and operational health states.
- **Load Balancing Action Console**: Command controls enabling manual edge rerouting or comprehensive reporting compilations.
- **Global Command Console**: Monospaced terminal outputting active system observer log scripts.

## 🎨 Theme Details

- **Palette**: Pitch Black (`#030408`), Surface Navy (`#0a0d16`), Success Emerald (`#10b981`), Cyan Accent (`#0ea5e9`), Warning Amber (`#f59e0b`).
- **Typography**: Google Fonts Inter and JetBrains Mono.
- **Animations**: Soft entrances, scale pulses, and interactive transitions.
