# Animated Login/Signup Form with Tab Switch

A smooth, animated authentication form component built with pure HTML and CSS.

## Features
- Animated tab switch between Login and Signup panels
- Sliding pill indicator using CSS `transform` and `transition`
- `fadeSlideIn` keyframe animation on panel switch
- Ripple effect on submit button click
- Glassmorphism card design with `backdrop-filter: blur`
- Focus states on inputs with glow effect

## Files
- `demo.html` — Interactive demo
- `style.css` — All styles

## Preview
Tab switches smoothly between Login and Signup with an animated gradient indicator sliding across.
