`clip-path-polygon-tm` — CSS utility

## Usage

```html
<link rel="stylesheet" href="submissions/examples/clip-path-polygon-tm/style.css">
<div class="clip-path-polygon-tm">
  <!-- Demo content -->
</div>
```

## Available Classes

- .clip-path-polygon-[triangle|diamond|pentagon|hexagon]

## Impact

Creates geometric shapes without images
