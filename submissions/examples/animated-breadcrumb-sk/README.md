# animated-breadcrumb-sk

Breadcrumb navigation with arrow separators between levels. Multi-level path displays with hover underline effect on clickable links. The current page is highlighted and non-clickable via `aria-current="page"`. The breadcrumb wraps gracefully on narrow screens.

**Features:**
- Arrow separators (›) between items
- Hover underline slide-in effect
- Current page highlighted (non-interactive)
- Responsive wrapping
- Accessible `aria-label` and `aria-current`
