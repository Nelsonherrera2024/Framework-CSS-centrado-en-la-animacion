# Animated Spinners

A collection of CSS-only loading spinners with smooth infinite animations.

## Features
- 5 spinner types: Circle, Dots, Ring, Pulse, Hourglass
- Smooth infinite CSS animations
- Customizable size and color via CSS variables
- Clean card-based demo layout
- Lightweight - no JavaScript required

## Spinner Types
- **Circle**: Classic rotating border spinner
- **Dots**: Bouncing dots with staggered delays
- **Ring**: Multi-colored border rotation
- **Pulse**: Expanding/contracting circle
- **Hourglass**: Alternating rotation effect

## Usage
Include `style.css` and use the BEM-structured HTML from `demo.html`.
