# Submissions

This directory contains community-submitted components, examples, and documentation improvements for EaseMotion CSS.

## Structure

```
submissions/
├── examples/        # UI component demos with raw CSS
│   └── your-feature/
│       ├── demo.html
│       ├── style.css
│       └── README.md
└── docs/            # Documentation improvements
    └── your-guide/
        ├── demo.html
        ├── style.css
        └── README.md
```

## Submission Workflow

1. Create a folder inside `submissions/examples/` or `submissions/docs/`
2. Include `demo.html`, `style.css`, and `README.md`
3. Open a Pull Request against the `main` branch
4. The maintainer reviews, standardizes, and integrates accepted submissions

See [CONTRIBUTING.md](../CONTRIBUTING.md) for full guidelines.
