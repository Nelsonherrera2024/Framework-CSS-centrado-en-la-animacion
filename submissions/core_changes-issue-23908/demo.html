ease-carousel

CSS-only carousel component using scroll-snap-type. Includes navigation dots, prev/next buttons, captions, and variants. No JavaScript required for scrolling — dots and buttons are enhanced with minimal JS for click navigation.

Usage

<!-- Basic carousel with dots and buttons -->
<div class="ease-carousel" id="my-carousel">
  <div class="ease-carousel-track">
    <div class="ease-carousel-slide">
      <img src="slide1.jpg" alt="Slide 1" />
    </div>
    <div class="ease-carousel-slide">
      <img src="slide2.jpg" alt="Slide 2" />
    </div>
  </div>

  <button class="ease-carousel-btn ease-carousel-btn-prev"
    onclick="scrollCarousel('my-carousel', -1)">&#8249;</button>
  <button class="ease-carousel-btn ease-carousel-btn-next"
    onclick="scrollCarousel('my-carousel', 1)">&#8250;</button>

  <div class="ease-carousel-dots" id="dots-my-carousel"></div>
</div>
