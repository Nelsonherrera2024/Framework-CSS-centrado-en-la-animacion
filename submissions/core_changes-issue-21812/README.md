# Lingva — Language Learning Platform Landing Page

1. **What does this do?**  
   A friendly, global-themed full-page landing page for a language learning SaaS platform, built entirely with EaseMotion CSS utility classes and component classes — no custom keyframes.

2. **How is it used?**  
   Open `demo.html` directly in any browser. The page links to EaseMotion CSS via CDN and uses only existing `ease-*` classes for all animations, layout, and interactions.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can build complete, production-quality EdTech landing pages with zero custom animations and no build step. This language-learning theme uses a warm teal/purple global palette.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Teal-purple gradient hero with floating flag emojis (`ease-float`), animated entrance (`ease-slide-up`, `ease-delay-*`), yellow highlight text |
| Features | 6 feature cards (personalized lessons, speaking practice, AI tutor, gamification, cultural immersion, progress tracking) |
| Course Catalog | 9 language cards with flags and learner counts in a responsive grid |
| Pricing | 3-tier (Free, Premium $12.99, Family $19.99) with "Most Popular" featured plan |
| Success Stories | 3 student cards with language achievement badges |
| App Download CTA | Full-width gradient CTA with iOS and Android download buttons |
| FAQ | 4-item native `<details>/<summary>` accordion with plus/rotate toggle |
| Footer | 4-column footer with languages, company, support links |
