# Button Loading State Demo

## 📌 Description
This demo shows a button loading state to improve user experience.

## ✨ Features
- Button shows loading text on click
- Button gets disabled during loading
- Prevents multiple clicks
- Improves UX

## 🚀 How it works
Click the button → it changes to "Loading..." → disables → resets after 2 seconds

## 🧪 Usage
Open `demo.html` in browser and click the button.
